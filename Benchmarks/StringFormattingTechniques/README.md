# String Formatting Techniques

This benchmark it to help determine the most appropriate pattern for creating formatted strings:
- `string.Format`
- `string` interpolation
- `string` addition operator `+`
- `StringBuilder`
- `string` add equals operator `+=`
- `string.Concat`
